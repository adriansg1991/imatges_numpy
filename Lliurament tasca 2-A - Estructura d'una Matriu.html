Mostreu-me amb exemples de diferents matrius, la regla fonamental de Broadcasting que diu : "les matrius es poden transmetre / broadcast si les seves dimensions coincideixen o si una de les matrius t una mida d'1".

El broadcasting s un procs automtic implementat en les operacions de NumPy per eficientar els clculs amb arrays multidimensionals. Es fa broadcasting encara que les matrius tinguin diferent forma, aix s, les seves dimensions han de coincidir o una de les matrius ha de tenir mida 1.

a = np.arange(6).reshape(2,3)
print(f'Matriu a:\n{a}\n')

b = np.array([0,1,2])

)
